Hosting Your Blog on GitHub Using Octopress

Jan 10th, 2013

Jekyll is a static site generator. It allows you to write your pages in Textile or Markdown which then gets compiled into HTML. Because no server side scripting or databases are involved, page load times are lightning fast. You can host these static HTML files on your own web server, Amazon S3, or even use GitHub Pages. I decided to go with GitHub Pages for this blog.
